OAuth2Android by Xotan

OAuth 2.0 library for Android clients supporting Bearer- and Mac- Tokens

This project intends to make the use of OAuth 2.0 easier on Android Clients. Even if OAuth2 is less complicated to use than OAuth1.a and it may seem unnecessary to use a library, this library still can take care of most OAuth specific aspects to help the developer concentrate on his app and not on how to use OAuth. This library automatically recognizes which type of Access-Tokens is used by the service provider and handles all requests accordingly.

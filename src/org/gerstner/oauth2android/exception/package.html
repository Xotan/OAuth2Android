This Package contains all OAuth specific Exceptions.
Every Exception in this package extends the OAuthException. OAuthExceptions have more parameters than a normal Exception. They can contain an error String, a more detailed error description and a url leading to a website with more detailed informations about the error. If those parameters are acutally present depends on the Exception and the server returning the error.
For more informations see {@link org.gerstner.oauth2android.exception.OAuthException}
